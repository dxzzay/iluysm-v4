Como ligar o bot pelo console

Bem se você estiver usando Windows, é algo simples. Basta clicar na basta do bot com o botão direito segurando as teclas CTRL + SHIFT. Logo irá abrir mais opções

Basta clicar em "Abrir uma janela de comando aqui." Caso esteja no Windows 10+, basta abrir um Power Shell na pasta.

Assim que clicar irá abrir uma CMD, basta digitar node . dentro dela.

ERRO DE INICIALIZAÇÃO

Bem com atualizações do Discord, algumas tokens não serão possiveis ligar, você verá um erro dentro da node_modules, então se não funcionar tente trocar de conta ou ligar em uma token de bot que não terá esse erro.
